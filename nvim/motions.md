# Custom Motions

## Treesj
- <space>m - toggle
- <space>j - join 
- <space>s - split

## Treesitter-textobjects
- `af` - Around function
- `if` - Inside function
- `ac` - Around class
- `ic` - Inside class
- `ai` - Around conditional
- `ii` - Inside conditional
- `al` - Around loop
- `il` - Inside loop
- `<leader>a` - Swap next parameter
- `<leader>A` - Swap previous parameter

## Conform
- No specific motions. Provides formatting on save.
- Automatic formatting for supported file types.
